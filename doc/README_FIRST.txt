This is a distribution of FRED that has been modified for Windows OS compatibility.
It has been tested on Windows 7 and 10.

Before running FRED it is important to change the Start menu shortcut of FRED to always run as Administrator.
To do this:
1. Click the Start button on the Windows taskbar to open the Start Menu
2. Open the folder labeled "FRED"
3. Right click on the FRED shortcut and click on "properties"
4. Click on "Advanced..."
5. Check the box next to "Run as administrator"
6. Click OK
7. Click Apply
8. Click OK to close the FRED Properties dialog

Now FRED can be run as administrator by just clicking on the FRED Start menu shortcut.
